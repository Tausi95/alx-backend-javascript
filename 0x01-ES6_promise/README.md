# ES6 Promises Project

Welcome to the ES6 Promises project! This project explores the functionality and features of JavaScript promises, showcasing how to handle asynchronous operations effectively.

## Table of Contents

- [Description](#description)
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)

## Description

This project includes various examples of how to work with promises in JavaScript, including resolving and rejecting promises, chaining multiple promises, and handling errors. It aims to provide a better understanding of asynchronous programming in JavaScript.

## Installation

To get started with this project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/tausi95/alx-backend-javascript.git
   cd alx-backend-javascript/0x01-ES6_promise
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the examples:
   ```bash
   npm run dev <your-main-file.js>
   ```

## Usage

- Each JavaScript file in the project demonstrates a specific use case for promises. You can modify and run them to see how promises work in practice.
- To run a specific file, replace `<your-main-file.js>` with the corresponding filename, e.g., `4-main.js`.
